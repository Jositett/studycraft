📘 Usage Guide

📄 Upload Document

Drag & drop or click to select your source file. Supported formats:

  • PDF — textbooks, papers, slides
  • DOCX — Word documents
  • EPUB — e-books
  • TXT / MD — plain text or Markdown
  • RTF — rich text format

✏️ Subject Name

Optional. If left blank, StudyCraft auto-detects the subject from your document content.

✅ Answer Key

When checked, a separate answer key file is generated alongside the practice guide with solutions to all quiz questions.

📎 Context Files

Add supplementary files (lecture notes, references) that get indexed into RAG for richer, more accurate content. These are not turned into chapters.

🤖 Model Selection

Choose an AI model from OpenRouter. Free models work well for most documents. Paid models produce higher quality for complex subjects.

Click the ↻ button to refresh the model list from the API (cached 24h).

🎨 Theme

Choose a color theme for all exported files (HTML, PDF, DOCX, EPUB). Dark is the default.

🎯 Difficulty

Controls the complexity of examples, exercises, and quiz questions:

  • Beginner — simple vocabulary, basic recall
  • Intermediate — multi-step reasoning, real-world scenarios
  • Advanced — production-grade, design & evaluation

📥 Output Formats

Once generated, download your guide in:

  • MD — Markdown source
  • HTML — styled web page
  • PDF — print-ready document
  • DOCX — Word format
  • EPUB — e-reader format

🎵 Audio Guide

Check the "Generate audio guide" box to produce a narrated version using Text-to-Speech. Built-in engines:

  • KittenTTS (default) — lightweight, CPU-only, 8 voices
  • Chatterbox — high quality, multilingual, MIT license
  • Coqui/XTTS-v2 — 1100+ languages
Install optional engines if needed: uv sync --extra tts-kitten or uv sync --extra tts.

🎬 Video Guide

Check "Generate video guide" to create a video presentation with AI-generated visuals and narration. Uses OpenRouter video models (free tier available). Install: uv sync --extra video. Experimental — may take several minutes.

💡 Tips

  • Larger documents produce more chapters
  • Well-structured docs (with headings) yield better outlines
  • Use context files for supplementary material
  • Free models have rate limits — generation may take a few minutes
  • Choose a theme that matches your subject's aesthetic
  • Difficulty level adapts example complexity and quiz depth
